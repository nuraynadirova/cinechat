# 🎬 CineChat — AI Film Assistant

CineChat is an AI-powered chatbot that helps users discover films, get recommendations, and find detailed movie information through a conversational interface.

## ✨ Features

- 🤖 AI-powered conversational film assistant
- 🎥 Movie recommendations based on user preferences
- 🔍 Film information and details on demand
- 💻 Clean and simple web interface

## 🛠️ Technologies Used

- **Python** — AI logic and backend
- **HTML** — Web interface
- **AI Integration** — Conversational response generation

## 🚀 How to Run

1. Clone the repository
```bash
   git clone https://github.com/nuraynadirova/cinechat.git
```
2. Install dependencies
```bash
   pip install -r requirements.txt
```
3. Run the app
```bash
   python main.py
```
